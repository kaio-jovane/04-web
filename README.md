# Atividade de Web - Site de Viagens

**Desenvolvedor:** Kaio Jovane
**Disciplina/Período:** Sistemas de Informação - 3º Período

## 📖 Sobre o Projeto

Esse website é focado em turismo desenvolvido como parte das práticas de programação front-end. O objetivo desse site é consolidar os conhecimentos de desenvolvimento web.

O site simula uma agência de viagens virtual, destacando cinco destinos turísticos do Brasil: Capitólio, Fernando de Noronha, Gramado, Porto Seguro e Rio de Janeiro. A interface foi projetada para ser semântica e visualmente agradável, sendo usado HTML, CSS e JavaScript, foi garantido que as imagens e os elementos se comportem perfeitamente e seja responsivo para diferentes tamanhos de tela.
